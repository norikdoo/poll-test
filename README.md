# poll-test cmdline tool

> poll-test cmdline tool for testing poll() syscall on arbitrary file.

## Setup

```shell
make
```

## Usage

```shell
usage: poll-test [-f file]
        [-m POLLMASK]
        [-o FOPEN_FLAGS]
        [-t timeout]
        [-l] [-i] [-e] [-w]
        [-h]
```
